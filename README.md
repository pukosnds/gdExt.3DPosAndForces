# gdExt.3DPosAndForces
Gdevelop's extension for forces and positions.


# what does this extension does?

adds to gdevelop functionalites for forces, put objects in front of another or in front of camera.
translates 3D position for 2D planes (useful for objective icons or compass)
adds distance constraint
and more!

# How do i ask an feature?

create an github account, go to the issues tab and simply do your feature request or report an bug!
bugs always will be on top and we will work on it faster as we can

you can contribute by creating an fork of this repository, add your changes (adding your version in the version folder for example, or your example projects, dadadad) and then you can do an pull request!
